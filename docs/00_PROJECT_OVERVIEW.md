# AI 감정 분석기 (AI Sentiment Analyzer)

## 목표 (Goal)
BMW M의 산업적 UI 철학에서 영감을 받은 영화 같은 느낌의 AI 감정 분석 웹 서비스를 구축합니다.

## 핵심 기능 (Core Features)
- 텍스트 입력 (Text input)
- AI 감정 분석 (AI sentiment analysis)
- 긍정 / 부정 / 중립 결과 (Positive / Negative / Neutral result)
- 신뢰도 퍼센트 (Confidence percentage)
- 분석 이유 (Analysis reason)
- 결과 모달 UI (Result modal UI)
- OpenAI API 연동 (OpenAI API integration)
- Supabase 로깅 (Supabase logging)

## 기술 스택 (Tech Stack)
- 프론트엔드: HTML, CSS, JavaScript
- 백엔드: Node.js + Express
- AI: OpenAI API
- DB: Supabase
- 배포: Vercel

## 사용자 흐름 (User Flow)
1. 사용자가 텍스트를 입력합니다.
2. 분석 버튼을 클릭합니다.
3. 백엔드 API 요청이 전송됩니다.
4. OpenAI가 감정을 분석합니다.
5. 모달에 결과가 표시됩니다.

## 성공 기준 (Success Criteria)
- 빠른 응답 속도
- 미니멀한 UI
- BMW M에서 영감을 받은 다크 경험
